# MEME决策平台v0.1
核心：情绪指标与价格行为的对齐分析
功能：将非结构化的时间点数据（社交信号）映射到结构化的时间序列数据（K线）上。
技术原理：
分解
价格数据层：通过 Web3 接口（如 DexScreener API、Birdeye 或直接监听 RPC）获取代币的 OHLCV（开盘、最高、最低、收盘、成交量）数据。
信号采集层：通过爬虫或 API 实时监控 X (Twitter) 或 Telegram 频道，记录下含有特定代币关键词/合约地址的消息及精确时间戳。
坐标映射层：将消息时间戳 $t_{social}$ 与 K 线的周期区间 $[t_{start}, t_{end}]$ 进行匹配。如果 $t_{social}$ 落在某个 K 线的时间段内，就在该 K 线的 High（最高点）上方或 Low（最低点）下方打一个标记。
